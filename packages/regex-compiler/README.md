# @regex-studio/regex-compiler

A multi-target translation compilation engine that transforms Regex Studio AST representations into standard string expressions compatible with major programming language platforms.

## Features

- **Multi-language Support**: Compiles AST trees to JavaScript, Python, Go, Rust, Java, PHP, C#, and Ruby patterns.
- **Syntactic Bridging**: Handles named group translations and language-specific delimiter layouts natively.
- **Escape Management**: Escapes characters automatically according to standard string specifications.

## Installation

```bash
npm install @regex-studio/regex-compiler
```
