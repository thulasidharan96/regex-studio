# @thulasidharan96/flow-engine

Visual canvas node and connection layout logic wrapping React Flow to manage interactive visual regular expression nodes.

## Features

- **Bidirectional Mapping**: Translates raw AST trees to logical visual canvas nodes and edges.
- **Node Configurations**: Declares customizable flow states for anchors, lookarounds, characters, and capture groups.

## Installation

```bash
npm install @thulasidharan96/flow-engine
```
