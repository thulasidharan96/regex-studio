# @regex-studio/regex-debugger

Interactive simulation engine that models step-by-step regex matching execution, backtracks, success, and failure branches directly in the client.

## Features

- **Step Generation**: Calculates precise evaluation ticks mapped directly to the AST.
- **Trace Logs**: Formats precise matching step outputs.
- **Timeline Milestones**: Flags crucial matching highlights (match, backtrack, start, success, fail).

## Installation

```bash
npm install @regex-studio/regex-debugger
```
