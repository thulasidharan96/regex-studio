# @regex-studio/regex-parser

An advanced, fault-tolerant regular expression parser that compiles standard regex strings into fully structured Abstract Syntax Trees (ASTs).

## Features

- **Lexical Analysis & Tokenization**: Deconstructs modern regular expressions into discrete syntactic segments.
- **Intelligent Error Recovery**: Recovers gracefully from typing errors (such as unclosed groups) with actionable suggestions.
- **Full Grammar Support**: Recognizes advanced patterns including capturing/non-capturing groups, lookarounds, and custom character classes.

## Installation

```bash
npm install @regex-studio/regex-parser
```
