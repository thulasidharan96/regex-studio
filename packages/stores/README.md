# @regex-studio/stores

Global React state management stores using Zustand, orchestrating canvas state, visual updates, parse feedback, and automatic background storage saving.

## Features

- **Document State Sync**: Coordinates visual canvas nodes with standard regex output.
- **Background Auto-Saves**: Seamlessly persists user environments onto IndexedDB.

## Installation

```bash
npm install @regex-studio/stores
```
