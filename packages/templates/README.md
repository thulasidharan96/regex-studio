# @regex-studio/templates

A pre-packaged library registry containing high-quality standard regular expression templates complete with pre-compiled AST visual layouts.

## Features

- **Standard Library Recipes**: Templates for email, phones, URLs, dates, IP addresses, JWT tokens, and passwords.
- **Accompanying Unit Examples**: Includes correct and incorrect testing strings.

## Installation

```bash
npm install @regex-studio/templates
```
