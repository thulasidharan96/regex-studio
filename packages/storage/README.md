# @thulasidharan96/storage

Robust client-side local offline persistence adapter wrapping IndexedDB schemas (via Dexie) for seamless, auto-saving document states.

## Features

- **IndexedDB Schema Adapters**: Automatically creates tables for projects, templates, and layouts.
- **Fast Transactions**: Synchronizes local state changes with complete reliability.

## Installation

```bash
npm install @thulasidharan96/storage
```
